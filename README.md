# Comp-Python-Final-Project
This is the repo for the final project for python.

to get everything up and running. If you are on a windows or non arch based linux system.
```
pip install -r requirements.txt
streamlit run src/app.py 
```

If you are on a Arch based linux distro you must first create an env then run the pip command
```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run src/app.py
```
